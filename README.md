# 🎬 Movie Success Analysis

## 📌 Project Overview
This project analyzes a dataset of 1000 movies to understand
what factors contribute to a movie's success.

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Key Questions
- Do higher ratings lead to higher revenue?
- Which directors produce the most successful films?
- Does runtime affect ratings?
- How do trends change over time?

## 📈 Key Insights
- High ratings do not guarantee high revenue
- Director reputation influences success
- Runtime has little impact on ratings
- Revenue is concentrated among a few top movies
